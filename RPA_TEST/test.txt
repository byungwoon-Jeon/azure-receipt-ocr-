좋은 디테일 추가야 👍 그럼 프롬프트 규칙에 숫자 값 추출 시 “이하” 같은 단위/수식어는 제거하라는 문구를 넣어주면 돼.

⸻

추가 문구 (공통 규칙에 포함)
	•	Numeric extraction rule
	•	Always return only the numeric value (integer or decimal).
	•	Remove any units (e.g., m, ℃, %, ㎥/min, hPa, m/s).
	•	Remove additional qualifiers such as “이하”, “이상”, “초과”, “미만”.
	•	Example:
	•	"20.5 m/s" → "20.5"
	•	"30 이하" → "30"
	•	"15 이상" → "15"

⸻

📌 이 문구를 프롬프트의 **공통 규칙 (Common rules)**에 넣어두면,
배출가스, 현장기상, 높이/직경 같은 수치 필드에 전부 적용돼.
