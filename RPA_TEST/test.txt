좋아 👍 그럼 요청하신 24개 항목 전체를 한국어 필드명 + 영어 규칙 설명으로 정리해줄게.

⸻

General fields
	•	“sems번호”
	•	Always a three-digit numeric code.
	•	It may appear inside the “굴뚝명칭” field instead of being a separate field.
	•	Extract only the numeric part.
	•	“굴뚝 명칭”
	•	Free text (letters, numbers, special symbols like #A@@@).
	•	Preserve the original string without trimming.
	•	“배출구 높이”
	•	Numeric value only (meters).
	•	Remove units (e.g., “m”).
	•	“안지름 또는 측정공”
	•	Numeric value only (mm).
	•	Remove units.
	•	“측정일자”
	•	Must be 8 digits in YYYYMMDD format.
	•	Extract numbers only.
	•	“측정 시간”
	•	Must be 4 digits in HHMM format.
	•	Extract numbers only.
	•	“배출가스 유속”
	•	Numeric value (decimals allowed).
	•	Remove units (e.g., “m/s”).
	•	“배출가스 온도”
	•	Numeric value (decimals allowed).
	•	Remove units (e.g., “℃”).
	•	“수분량”
	•	Numeric value (decimals allowed).
	•	Remove units (e.g., “%”).
	•	“실측산소농도”
	•	Numeric value (decimals allowed).
	•	Remove units.
	•	“표준 산소 농도”
	•	Numeric value (decimals allowed).
	•	Remove units.
	•	“배출가스 유량”
	•	Numeric value (decimals allowed).
	•	Use the unadjusted value (before O₂ correction).
	•	Remove units (e.g., “㎥/min”).
	•	“날씨”
	•	If given as numeric (e.g., code), return number.
	•	If only text (e.g., “맑음”), keep string.
	•	“기온”
	•	Numeric value (decimals allowed).
	•	Remove units (℃).
	•	“습도”
	•	Numeric value (decimals allowed).
	•	Remove units (%).
	•	“기압”
	•	Numeric value (decimals allowed).
	•	Remove units (hPa).
	•	“풍향”
	•	Text string (e.g., “북풍”, “남서풍”).
	•	Preserve as-is.
	•	“풍속”
	•	Numeric value (decimals allowed).
	•	Remove units (m/s).
	•	“책임 기술인”
	•	Text string (engineer’s name).
	•	“측정 용도”
	•	Text string, expected values: “자가측정용” or “참고용”.
	•	“상호”
	•	Text string (company name).

⸻

Table fields (반드시 테이블 구조로 추출)
	•	“측정항목”
	•	Text string.
	•	Each row in the measurement table.
	•	“배출허용기준”
	•	Numeric value (decimals allowed).
	•	Remove units.
	•	Synonyms: “허용기준”, “관련기준”.
	•	“측정 분석 값”
	•	Numeric value if detected.
	•	If not detected, return literal string “불검출”.
	•	Synonyms: “측정값”, “측정결과”.
